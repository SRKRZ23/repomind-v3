39
CUDA→HIP rules + kernel-launch rewrite
How It Works
The agent finds CUDA files and calls the migration tool, which applies 39 API-mapping rules (memory/stream/event/device/error + math-library headers) and rewrites kernel-launch syntax (kernel<<<grid,block>>> → hipLaunchKernelGGL), emitting an appliable unified diff (git apply-clean).

Unit-tested against demo/cuda_sample.cu. Complements AMD HIPIFY.
Why AMD Wins This
50M+ lines of CUDA code on GitHub. The #1 barrier to AMD GPU adoption is migration friction. REPOMIND removes it automatically. Roadmap: AI-guided migration for non-trivial CUDA patterns (custom kernels, PTX).
